# Vaidic Soni Portfolio

Personal portfolio website built with HTML, CSS, and Vanilla JavaScript. 

## Tech Stack
* HTML5
* CSS3 (Custom properties, CSS Grid/Flexbox)
* Vanilla JavaScript (ES6)
* Devicon (Icons)

## Setup & Running Locally
1. Clone the repository.
2. Open the project root folder in VS Code.
3. Install the **Live Server** extension.
4. Click **Go Live** or right-click `index.html` -> **Open with Live Server**.

## File Structure
* `index.html`: Main entry point.
* `css/`: Modular stylesheets (`variables.css`, `base.css`, `projects.css`, `responsive.css`, etc.).
* `js/`: Modular scripts (`particles.js`, `skills.js`, `nav.js`, `reveal.js`).
